## InputLabel

Libellé dans une ligne de saisie

### Props

#### Obligatoires

| Nom   | Type   | Commentaire |
| ----- | ------ | ----------- |
| label | Chaine | Libellé     |

#### Optionnels

| Nom     | Type   | Remarque | Commentaire              |
| ------- | ------ | -------- | ------------------------ |
| size    | Chaine | null     | Taille                   |
| htmlFor | Chaine |          | Identifiant du champ lié |

### Exemple

```
<InputLabel
  label="Nom"
/>
```
